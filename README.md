## SETUP
Please add .evn following:
```sh
PORT=3001
DB_URI=@value1
OWNER_PRIVATE_KEY=@value2
```
With:
* @value1: URI to connect to mongodb
* @value2: Private key of the owner, who deploys the contracts.
